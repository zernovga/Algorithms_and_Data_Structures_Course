## Задание 8

Определите колчиество операций суммирования и асимптотику.

```python
def task8(values: list, size: int):
    sum = 0
    j = 0
    for i in range(0, 1000):
        sum += i
    for i in range(0, size):
        j = 1
        while j <= size:
            sum += values[j-1]
            j *= 2
    return sum
```

ассимптотика: O(n log n)
операций суммирования: 1000+n log n
