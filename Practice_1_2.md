## 8. Найти сложность приведенной ниже программы: 

```python
def function(n):
    i = 1
    s = 1
    while (s < n):
        s = s + i
        i+=1
```
В функции используется только один цикл while, который будет выполняться до тех пор, пока значение переменной s не станет больше или равным n. В каждой итерации цикла мы увеличиваем значение переменной i на 1 и прибавляем его к переменной s. 
Цикл будет выполнен n - 1 раз, так как на последней итерации значение переменной s уже будет больше или равно n. 
сложность - O(n).

## 10. Найти сложность приведенной ниже программы в лучшем и в худшем случае: 


```python
def fun(a, b):
    while (a != b):
        if (a > b):
            a = a - b
        else:
            b = b - a
```             
Лучший случай: a и b равны друг другу или одно из них равно нулю. Сложность алгоритма будет O(1).
Худший случай: a и b являются взаимно простыми числами. В этом случае алгоритм будет выполняться до тех пор, пока одно из чисел не станет равным нулю. Сложность алгоритма в худшем случае будет O(log n), где n - меньшее из двух чисел a и b.
