## 4. Найти сложность приведенной ниже программы:

```python
def funct(n):
    count = 0
    for i in range(n//2, n+1):
        j = 1
        while j <= n:
            j = 2 * j
            k = 1
            while k <= n:
                k = 2 * k
                count += 1
```
В цикле for сложность n так как константа 2 откидывается. Все эти циклы зависимы поэтому они перемножаются. Следущие два цикла похожи и они имеют сложность log(n) и при их перемножении получается log^2(n)
Otv:O(n*log^2(n))
