## 10. Найти сложность приведенной ниже программы в лучшем и в худшем случае: 


```python
def fun(a, b):
    while (a != b):
        if (a > b):
            a = a - b
        else:
            b = b - a
```             
лучший: O(1), когда a и b равны друг другу или одно из них равно 0
худший: O(log(n)), где n меньшее из двух чисел a и b
