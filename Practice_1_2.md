## 4. Найти сложность приведенной ниже программы:

```python

    count = 0
    for i in range(n//2, n+1):
```

## 5. Найти сложность приведенной ниже программы: 

```python

    count = 0
    for i in range(n//2, n+1):
        j = 1

            j += 1
```
