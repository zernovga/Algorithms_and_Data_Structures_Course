какой-то список
его длина = x
максимальный элемент = первый элемент списка
какое то число f = 1

пока f меньше x:
    если максимальный элемент меньше чем элемент с позицией f
    то максимальный элемент = этот элемент с позицией f
    f += 1

максимальный элемент = какой то элемент списка, найденный циклом

Свойства:
1. Конечность. Алгоритм закончится, когда f станет не меньше длины массива
2. Определенность. Все действия недвусмысленны
3. Ввод. Алгоритм принимает массива
4. Вывод. Алгоритм выводит максимальный элемент массива
5. Эффективность. Алгоритм сработает так быстро, как это возможно. Скорость выполнения зависит от количества элементов массива.
